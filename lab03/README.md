# Лабораторная работа 03 — Компоненты React и хуки

## Как запустить
npm install
npm run dev

## Структура
Задача 1 — Счетчик шагов с props и state
Задача 2 — Профиль пользователя с useEffect и API fetch

## Объяснение
Props передаются из родительских компонентов и доступны только для чтения.
State управляется внутри компонентов, и обновления приводят к перерисовке.
